RunRepeat.com is a review aggregator of running shoes. This plugin allows you to add our reviews to your product pages or review pages.

== Description ==
[RunRepeat.com](http://runrepeat.com/ "RunRepeat.com - review aggregator of running shoes") is a review aggregator of running shoe reviews. We are 100% independent, and do not review running shoes ourselves.

Runners visit the website to read reviews, write their own reviews, check ranking lists and to compare current/new running shoes with similar models.
This plugin allows you to add our reviews of running shoes to your pages. Simply install the plugin and insert the shortcode [**runrepeatratings**] to the pages where you want to display reviews.

**How it works:**

*   Insert the standard shortcode [**runrepeatratings**]
*   The plugin automatically detect what product is displayed
*   Ratings from RunRepeat is inserted automatically

Learn more about the benefits of this by visiting [our B2B page](http://runrepeat.com/b2b), where you will find detailed documentation on the effect of inserting our reviews to your pages.

== Installation ==

**From your WordPress dashboard**

*   Visit "_Plugins > Add New_"
*   Search for "_RunRepeat.com Rating Solution_"
*   Activate RunRepeat.com Rating Solution from your Plugins page.

**From WordPress.org**

*   Download _RunRepeat.com Rating Solution_.
*   Upload the "_runrepeatcom-rating-solution_" directory to your "_/wp-content/plugins/_" directory, using your favorite method (ftp, sftp, scp, etc...)
*   Activate *RunRepeat.com Rating Solution* from your Plugins page.

**Once Activated**

Go to _Tools > RunRepeat Solutions_ and here you can find more info about how to use the shortcode
